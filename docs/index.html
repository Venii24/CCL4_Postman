Creative Code Lab 4 (Group 2)

Deliver the mail and catch the train before time runs out!

Join Charlie on his daily journey across several unique islands to deliver mail. Meet Bruno, Daisy, and Kai, and help them stay connected! Deliver their letters, collect post stamps, and catch the Post Express before it leaves without you. Immerse yourself in a game world with charming aesthetics and a cozy ambiance.

Input

WASD for Player Movement

Left and Right Arrow Key to Rotate the Camera

F to deliver letter to NPCs and start Dialog

GameObjects that are never destroyed:

  • GameManager (manage Overlays and Score)
  • OverlayCanvas (Overlays for Winning, Dialog and Time Over)
  • DialogManager (to start the Dialog)
  • LevelLoader (for the Loading Screen between Scenes)

Interactable Objects:

  • Stamp Collectables 
  • Boxes (for the player to push)
  • LoadSceneTrigger (to stop the timer and start the train exit animation)

Additional Features

  • Dialog saved in JSON files and can be loaded in real time
  • Timer (Player has 5 Minutes for each level)
  • move Boxes out of the way to pass or place them right to be able to jump on higher platforms
